Welcome to Dueling Bandit Toolkit's documentation!
=================================================

A Python package for preference-based online learning with dueling bandits. Implements algorithms like Double Thompson Sampling, PARWiS, and Contextual PARWiS, with support for real-world datasets and evaluation metrics.
